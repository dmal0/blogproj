App Overview

This app is a basic blogging platform for sharing text and images.
Each Blog has one Author and multiple Posts, while each Post belongs to one Blog and can have multiple Comments.

The navigation bar at the top has a link to this page ("Index"), which turns into the Dashboard when logged in.
The navigation bar also has links to lists of all Blogs, Posts, and Comments, as well as a dropdown menu with links to, when logged in, the user's own Blog, their Post management page, their account (Author) settings, their Blog settings, and a log out link.
For logged-out users, only a registration and log in link are displayed.

The list for all Blogs does not contain "Edit" (Update) or "Delete" buttons because users should only be able to manage their own blogs, which is done via their individual Blog settings pages. In the list of all Posts, users are only shown the edit and delete buttons for their own Posts. The same applies to the list of all Comments.
