App Overview

This app is a basic blogging platform for sharing text and images.
Each Blog has one Author and multiple Posts, while each Post belongs to one Blog and can have multiple Comments.
In the final version of the app, this page will likely become a login page.

The navigation bar at the top has a link to this page ("Index"), the Dashboard, lists of all Blogs, Posts, and Comments, as well as a dropdown menu with links to the user's own Blog, their Post management page, their account (Author) settings, and their Blog settings. There is also a placeholder link for a lougout button, but it is disabled.

The list for all Blogs does not contain "Edit" (Update) or "Delete" buttons because users should only be able to manage their own blogs, which is done via their individual Blog settings pages. In the final version of the app, this should apply to the list of all Posts as well. The final version of the list of all Comments will also ideally only show the comments on the logged-in user's Blog Posts.

In this iteration of the app, there is also no way to create a new Blog or Author (which combined should, in some way, form an Account in the final version of the app) because we have not yet learned in class about authentication or managing accounts.
